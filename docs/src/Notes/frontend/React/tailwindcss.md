# tailwindcss

## pnpm 安装


pnpm下载：

`pnpm add -D tailwindcss@3.4.1 postcss autoprefixer`

::: warning
使用`-D tailwindcss postcss autoprefixer`下载版本为4.0.12，会在`npx tailwindcss init -p`中报错
:::

pnpm初始化Tailwind：

`pnpm exec tailwindcss init -p`
